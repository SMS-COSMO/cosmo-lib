# 示例API
示例api, 展示如何写一个api, 以及如何使用它
# 使用示例
```ts
import { sampleAPI } from "cosmo-lib";
console.log(sampleAPI('mixin')); // 输出: `Hello, mixin! Welcome to COSMO!`
```
